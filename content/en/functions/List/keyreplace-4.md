# List.keyreplace/4

{{< signature >}}

Receives a list of tuples and replaces the tuple whose `index`-th element is `key` with `new_tuple`.

{{< figure src="images/functions/List/keyreplace-4.svg" >}}

If no tuple is found the list is returned untouched.
